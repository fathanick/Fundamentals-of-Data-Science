# Introduction to Python for Data Science

This module introduces undergraduate Informatics students to Python for Data Science using Google Colab. It covers core libraries (NumPy, Pandas, Matplotlib, SciPy), probability distributions, and basic statistics through hands-on coding exercises.

## Topics Covered

**Notebook 1 — Python for Data Science**
- Google Colab basics (running cells, importing libraries, saving/sharing)
- NumPy (array creation, statistics, random data generation)
- Pandas (DataFrames, data exploration with `.shape`, `.describe()`)
- Matplotlib (histograms, bar charts, scatter plots)
- Exploratory data analysis with the Seaborn Tips dataset
- Student assignment: EDA on Penguins, Iris, or MPG dataset

**Notebook 2 — Probability & Statistics**
- Basic probability (sample spaces, events, complements, Law of Large Numbers)
- Discrete random variables: Bernoulli, Binomial, Poisson distributions
- Continuous random variables: Normal distribution, PDF, CDF, area-under-curve
- Descriptive statistics: mean, median, mode, standard deviation
- Grouped frequency tables with mathematical formulas
- Student assignment: simulate a distribution and compare theoretical vs empirical statistics

## Prerequisites
- Basic Python knowledge (variables, loops, functions)
- Google account (for Colab)
- Notebook 2 requires completion of Notebook 1

## Duration
Approximately 2–4 hours (1–2 hours per notebook)
