# Introduction to Python for Data Science

This module introduces undergraduate Informatics students to Python for Data Science using Google Colab. It covers core libraries (NumPy, Pandas, Matplotlib), random variables, and basic statistics through hands-on coding exercises.

## Notebooks

| # | Notebook | Description | Open in Colab |
|---|----------|-------------|---------------|
| 1 | `tutorial_random_variables_statistics.ipynb` | Introduction to random variables, basic statistics (mean, median, std), and data exploration with the Tips dataset | [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/fathanick/Fundamentals-of-Data-Science/blob/main/Introduction%20to%20Python%20for%20Data%20Science/tutorial_random_variables_statistics.ipynb) |

## Topics Covered
- Google Colab basics (running cells, importing libraries, saving/sharing)
- NumPy (arrays, random data generation)
- Pandas (DataFrames, data exploration)
- Matplotlib (histograms, bar charts, scatter plots)
- Discrete vs continuous random variables
- Mean, median, and standard deviation
- Case study with the Seaborn Tips dataset
- Student assignment with alternative datasets

## Prerequisites
- Basic Python knowledge (variables, loops, functions)
- Google account (for Colab)

## Duration
Approximately 1–2 hours
