# 📂 Fundamentals of Data Science

This repository contains all the Python code, Jupyter notebooks, datasets, and resources used for the **Fundamentals of Data Science** course. It serves as a practical, hands-on companion to learning core concepts in data analysis, visualization, machine learning, and more.

## 📚 Course Modules

| # | Module | Description | 
|---|--------|-------------|
| 0 | [Introduction to Python for Data Science](./Introduction%20to%20Python%20for%20Data%20Science) | Google Colab setup, NumPy, Pandas, Matplotlib basics, and introductory statistics |
| 1 | [Random Variables](./Random%20Variables) | Discrete random variables, PMF, probability distributions |
| 2 | [Statistics](./Statistics) | Statistical methods and analysis |
| 3 | [Data Visualization](./Data%20Visualization) | Visualization techniques and best practices |
| 4 | [Dimensionality Reduction](./Dimensionality%20reduction) | PCA and dimensionality reduction techniques |
| 5 | [Logistic Regression](./Logistic%20Regression) | Logistic regression for classification |
| 6 | [Naive Bayes](./Naive%20Bayes) | Naive Bayes classifier |
| 7 | [Decision Tree](./Decision%20Tree) | Decision tree algorithms |

## 🚀 How to Use

1. Click the **Open in Colab** badge on any notebook to run it directly in Google Colab
2. Or clone this repository:
   ```bash
   git clone https://github.com/fathanick/Fundamentals-of-Data-Science.git
   ```
3. Open notebooks in Jupyter or VS Code

## 📋 Prerequisites

- Basic Python programming knowledge
- Google account (for Google Colab)
- No prior Data Science experience required for the introductory module
